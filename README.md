# Unity_TimeScaleEditorTool

<img src="https://github.com/XJINE/Unity_TimeScaleEditorTool/blob/main/screenshot.png" width="100%" height="auto" />

Add GUI to edit Time Scale next to Play buttons.
This is same as ``Project Settings > Time > Time Scale``.

## Importing

You can use Package Manager or import it directly.

```
https://github.com/XJINE/Unity_TimeScaleEditorTool.git?path=Assets/TimeScaleEditorTool
```

NOTE: Because of the limitation of UnityPackageManager, it needs to install dependencies first.

### Dependencies

This project use following resources.
