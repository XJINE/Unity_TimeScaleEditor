# Unity_TimeScaleEditor

<img src="https://github.com/XJINE/Unity_TimeScaleEditor/blob/main/Screenshot.png" width="100%" height="auto" />

Add GUI on the right side of Play buttons to edit Time Scale.
This is same as ``Project Settings > Time > Time Scale``.

## Importing

You can use Package Manager or import it directly.

```
https://github.com/XJINE/Unity_TimeScaleEditor.git?path=Assets/TimeScaleEditor
```

NOTE: Because of the limitation of UnityPackageManager, it needs to install dependencies first.

### Dependencies

This project use following resources.

- https://github.com/marijnz/unity-toolbar-extender
